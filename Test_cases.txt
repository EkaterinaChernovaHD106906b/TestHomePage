Регистрация нового пользователя.
Шаги по воспроизведению:
1. Открыть страницу http://localhost:5000/. Ожидаемый результат: страница открывается.
2. Нажать на кнопку "Sign Up". Ожидаемый результат: появляется форма регистрации.
3. В поле "Email" ввести адрес электронной почты. Ожидаемый результат: система проводит валидацию, поле заполняется.
4. В поле "Password" ввести пароль. Ожидаемый результат: система проводит валидацию, поле заполняется.
5. Нажать кнопку "Sign Up". Ожидаемый результат: переход на страницу "Login".
Вход пользователя со страницы "Login".
Шаги по воспроизведению:
1.  Открыть страницу http://localhost:5000/. Ожидаемый результат: страница открывается.

7. В поле "Your Password" ввести пароль из пункта 4. Ожидаемый результат: система проводит валидацию, поле заполняется.
8. Нажать на кнопку "Login". Ожидаемый результат: переход на страницу "Profile"
